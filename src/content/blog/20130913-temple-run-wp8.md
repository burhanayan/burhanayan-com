Sizlerin de bildiği gibi, günümüz akıllı telefon teknolojisinde kullanılan yaygın cihazlara alternatif olarak güçlü Windows Phone 8’li cihazların birkaç ay önce piyasaya çıkmasıyla, kullanıcıların Marketplace”den beklentileri bir hayli arttı. Büyük yazılım şirketleri, diğer işletim sistemlerine ek olarak, uygulamalarını Windows Phone 8”e de adapte etmeye başladılar.

[![](http://i.imgur.com/In7le7k.jpg)](http://i.imgur.com/In7le7k.jpg)

Imangi Studios tarafından yayınlanan Temple Run artık Windows Phone 8’li cihazlar için indirelibilir durumda. 23 MB’lık büyüklüğe sahip oyun, tüm WP8’li cihazlar tarafından oynanabildiği belirtilirken, indiren kullanıcıların eleştirilerini okuduğumuzda, Nokia Lumia 620’de bellek yetersizliği bakımından oynanamadığı belirtiliyor. Ayrıca Nokia Lumia 920 kullanıcısı olarak, arada sırada el hareketlerini algılamama gibi problemlerle karşılaşıyorum. Sanırım yapılacak olan bir güncelleme, bu problemin üstesinden gelecektir.

Ayrıca medyaya sızan dedikodulara göre Temple Run 2 ‘nin önümüzdeki mayıs ayında çıkması bekleniyor..
